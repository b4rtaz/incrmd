# CalcCLI

The project is a simple CLI calculator implemented in JavaScript/Node.js. The calculator accepts 3 inputs:

- The first number
- The operator
- The second number

The calculator supports the following operators: +, -, \*, /, mod, pow.

Example usage:

```
$ node calculator.js -- 10 + 5
15

$ node calculator.js -- 10 - 5
5
```
